# New College Poker Society — Sponsorship Deck

A minimal static website for the New College Poker Society (NCPS) at the University of Oxford, used to share information about the society and attract sponsors.

## Live Site

[https://aaronrose227.github.io/NCPS/](https://aaronrose227.github.io/NCPS/)

## Built With

- HTML
- CSS

## Pages

- **Home** — Society name and navigation
- **About** — What we do and who we are
- **Sponsor Us** — Sponsorship information with an interactive poker table showing current and open sponsor seats
- **Committee** — Society committee members and roles
- **Contact** — How to get in touch
